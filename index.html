Graphical Programming Demo

Here we present the current state of the EduFill graphical programming system. The system is based on the Blockly framework available at Blockly Homepage Blockly is a framework designed to simplify the creation of web-based graphical programming tools. Blockly itself only provides the necessary infrastructure for building programming systems and is target language independent. For more information visit the Blockly homepage.

For EduFill's use, we made some changes to Blockly and added EduFill specific blocks. The whole EduFill Software toolbox is composed of three parts:

  • The actual programming system, presented on this page
  • The Python modules implementing the glue code to combine Blockly blocks into a working application
  • The functional software components implementing the actual functionality of our EduFill specific Blockly blocks, like 3D object recognition, base navigation, trajectory planning etc.
The last two parts make up the EduFill runtime environment and form a major part of the EduFill software toolbox.
